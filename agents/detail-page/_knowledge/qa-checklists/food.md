# 식품 카테고리 QA 체크리스트

식품(과일·신선·가공) 카테고리 상세페이지의 출고 기준.

## PASS 기준
- 전체 score ≥ 80
- High severity 이슈 0개
- 아래 필수 체크 모두 통과

---

## 필수 체크 (모두 PASS여야 함)

### A. 시각 통일성

- [ ] 페이지 전체 BG 종류가 3개 이하 (메인 + 강조 1~2)
- [ ] 카드 라디우스 16px 통일 (썸네일은 12px 일관)
- [ ] Eyebrow 라벨이 모든 섹션에서 같은 사이즈/스타일 (200×36 또는 240×36 표준)
- [ ] H1 크기 일관 (히어로 36px, 일반 섹션 28px)
- [ ] 사진 모서리 라디우스 통일

### B. 컬러 시스템

- [ ] 액센트 컬러는 단일 (또는 명확한 2색 시스템)
- [ ] 배경 컬러가 카테고리 톤 (식품 = 따뜻한 자연 톤, 다크/네온 X)
- [ ] 흰 텍스트는 대비 충족되는 진한 배경에만 사용
- [ ] 채도가 균형 (한 컬러만 튀지 않음)

### C. 카피 정확성

- [ ] 오타·맞춤법 0건
- [ ] 종결어미 일관 (~합니다 또는 ~해요 한 가지)
- [ ] 사장 메모리 규칙 위반 없음:
  - [ ] "특허" / "100%" / "유일" / "최고 효능" 같은 단정 없음
  - [ ] 산지·재배 조건 단정 없음
  - [ ] 검증 안 된 비교 ("1등", "1위" 등) 없음
- [ ] 감각 묘사는 일반화 OK (촉촉한·아삭한·은은한 등)

### D. FBM 후크 분포

- [ ] Pleasure spark 섹션 존재 (Hero 또는 Taste)
- [ ] Fear of Loss 섹션 존재 (제철·한정·재고)
- [ ] Hope 섹션 존재 (장면 시뮬레이션)
- [ ] Facilitator 존재 (고민 절약 메시지 — Brain Cycles)
- [ ] Kairos 섹션 존재 (지금 결제 신호)

### E. 사진 다양성

- [ ] 같은 사진 중복 사용 0건 (히어로와 클로즈업 둘 다 같은 컷 X)
- [ ] 카테고리 필수 컷 모두 존재:
  - [ ] 식욕 자극 클로즈업 (히어로용)
  - [ ] 풀스크린 구성샷 (매크로)
  - [ ] 단면 사진 (Taste)
  - [ ] 산지·포장 (Point 또는 Gallery)
- [ ] 좋은 컷이 작은 썸네일로 묻히지 않음 (예: 종이컵 크기 비교는 풀와이드여야)
- [ ] 갤러리 6컷 모두 다른 사진

### F. 정보 완결성

- [ ] 가격 명시 (옵션별)
- [ ] 배송 안내 명시 (출고 시각·택배사·예외 지역)
- [ ] 후숙·보관 가이드 (필요 카테고리)
- [ ] 농산물 표시사항 슬롯 또는 정책 섹션
- [ ] 사장님/판매자 신뢰 요소 (인용·서명·로고)

### G. 레이아웃

- [ ] 좌우 마진 일관 (40px 표준 또는 60px)
- [ ] 섹션 간 호흡 충분 (다음 섹션과 붙어있지 않음)
- [ ] 풀와이드 사진 후 여백 있음
- [ ] 글자 크기 모바일 가독 (본문 14px+)
- [ ] CTA 카드가 페이지 하단에 위치

## 권장 체크 (점수 가산)

### H. 임팩트

- [ ] 거대 타이포 (히어로 36px+) 임팩트 있음
- [ ] Yellow highlight band 1~2회 절제된 사용
- [ ] 인용문 (italic serif) 1~2회 감성 마무리
- [ ] 단면 컬렉션 같은 큰 사진 모먼트 1~2회

### I. 한국 시장 컨벤션

- [ ] POINT 01·02·03 같은 번호식 구조 (스마트스토어 컨벤션)
- [ ] 농가 직거래 / 산지직송 키워드 자연스럽게 노출
- [ ] 신선식품 환불 정책 명시 (외부 방치 시 환불 불가 등)
- [ ] 제주·도서산간 안내

### J. 시각 리듬

- [ ] 텍스트 무거운 섹션 후 풀스크린 사진 (호흡)
- [ ] 같은 BG가 5섹션 이상 연속 X
- [ ] CTA 메가카드 외엔 다크/풀블록 강조 X

## 카테고리별 추가 체크

### 과일 (특히 후숙과)
- [ ] 후숙 가이드 섹션 존재
- [ ] 보관 방법 명시 (실온 → 냉장)
- [ ] 후숙 확인법 (꼭지 말랑 체크 등)

### 가공식품
- [ ] 원재료·영양 정보 섹션
- [ ] 알레르기 정보
- [ ] 유통기한·소비기한

### 한우·축산
- [ ] 등급 표시 (1++, 1+ 등)
- [ ] 부위·조리법
- [ ] HACCP 인증 (있으면)

### 건강기능식품
- [ ] 기능성 인증 표시
- [ ] 섭취량·주의사항
- [ ] "효능" 단정 절대 금지

## 회송 결정 매트릭스

| 이슈 유형 | 회송 단계 | 예시 |
|---------|---------|------|
| 단정 키워드 발견 | copywriter | "100% 유기농" |
| FBM 후크 누락 | copywriter | Fear 섹션 없음 |
| 톤 불일치 | copywriter | "~합니다"와 "~예요" 혼용 |
| 컬러 시스템 깨짐 | visual-director | 4가지 BG 컬러 사용 |
| 카드 라디우스 불일치 | visual-director | 16px와 8px 혼용 |
| 사진 좌표 어긋남 | designer | 카드 밖으로 사진 튀어나감 |
| SVG 렌더 오류 | designer | 텍스트 깨짐·정렬 X |
| 빌드 실패 | designer | 파일 저장 실패 |

## 자동 점수 산정 가이드

```
필수 체크 (A~G): 각 영역 만점 10점, 총 70점
권장 체크 (H~J): 각 영역 5점, 총 15점
카테고리별 추가: 5점 보너스
사용자 컨펌 시 별도

PASS line: 80점 / FAIL: 79점 이하
```

- "이 카테고리는 X 사진이 항상 부족함" → benchmarks/{category}.md에 가이드 추가
- "Y 카피가 자주 단정으로 흘러감" → copy-patterns.md에 금지 키워드 추가
