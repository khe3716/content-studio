# 카피 패턴 라이브러리

## 1. FBM (Fogg Behavior Model) — 우리 메인 프레임워크

### 공식
```
B = M × A × T
Behavior = Motivation × Ability × Trigger
```
구매 행동이 일어나려면 동기·간편함·트리거가 동시에 충족돼야 함.

### Motivation (동기) 3축
1. **Pleasure / Pain** — 즉시 감각 (식욕·고통)
2. **Hope / Fear** — 미래 예측 (기대·손실회피)
3. **Social Acceptance / Rejection** — 사회적 인정 (가족 반응·인정)

### Ability (간편함) 6요소
1. Time (시간)
2. Money (돈)
3. Physical Effort (육체적 노력)
4. **Brain Cycles (생각해야 함)** ← 한국 시장 가장 큰 장벽
5. Social Deviance (사회적 일탈)
6. **Non-Routine (안 해본 것)** ← 새 상품의 장벽

### Trigger 3종
1. **Spark** — 동기 부족 사용자에게 동기 발화
2. **Facilitator** — 간편함 부족 사용자에게 진입 장벽 제거
3. **Signal** — 동기·능력 다 있는 사용자에게 적시 알림

### 섹션별 매핑 (스타일_1 메론 케이스)
```
hero       → Spark (Pleasure: 식욕 자극 큰 사진 + 꿀향 카피)
hook fear  → Spark (Fear: 5-6월 끝나면 내년까지)
hook hope  → Spark (Hope: 가족 반응 시뮬레이션)
taste      → Spark (Pleasure 심화: "설명 안 됨")
macro      → Visual Evidence (간접 Spark)
reasons    → Facilitator (4가지 이유로 신뢰 = brain cycle 절약)
point01    → Facilitator (산지 신뢰 — 의사결정 단순화)
point02    → Facilitator (고민 NO — 골라드림)
cuts       → Spark (Pleasure visual)
point03    → Ability (후숙 능력 부여)
kairos     → Signal (지금이 그 순간)
cta        → Signal (행동)
```

## 2. AIDA — 일반 광고 프레임워크

```
A: Attention (주목)
I: Interest (관심)
D: Desire (욕망)
A: Action (행동)
```

### 적용 시점
- 신상품 런칭
- 충동 구매 카테고리 (패션·잡화·간식)

### 단점
- "한국 신선식품" "고관여 제품"엔 약함 — Trust + Education 필요

## 3. PAS — 문제·위협·해결

```
P: Problem (문제 제기)
A: Agitate (불안 증폭)
S: Solution (해결 = 우리 상품)
```

### 적용 시점
- 헬스·건기식 (몸 관리 필요)
- 청소·정리 (집안 문제)
- 보안·안전

### 주의
- **식품엔 부적합** (불안 마케팅 = 신뢰 손상)
- 한국 시장은 PAS가 종종 거부감 일으킴

## 4. 4U — 광고 카피 4요소

```
Urgent (긴급)
Unique (유일)
Useful (유용)
Ultra-specific (구체적)
```

### 적용
- CTA·KAIROS 섹션
- "5-6월 한정 + 4종 중 2~3종 + 지금 출고" = 4U 충족

## 5. FAB — 기능·이점·혜택

```
Feature (사양/특징)
Advantage (그게 왜 좋은가)
Benefit (사용자가 얻는 것)
```

### 적용
- 기술 카테고리 (전자·디지털)
- POINT 섹션 본문에 적합

### 예시 (메론 POINT 01)
- F: 경북 고령 산지직송
- A: 중간 유통 0단계
- B: 수확 후 24시간 내 도착 — 가장 신선한 당도

# 카테고리별 권장 톤

## Food (식품)
- **프레임워크**: FBM (Pleasure + Fear + Hope + Facilitator + Kairos)
- **톤**: 따뜻한 + 정중한 + 체험형
- **종결어미**: ~합니다, ~드립니다 (정중)
- **금지**: 효능 단정·산지 단정·인공 vs 자연 대립
- **권장**: 감각 묘사·계절·산지 이름·관찰 가능 사실

## Electronics (전자)
- **프레임워크**: FAB + 4U + AIDA
- **톤**: 강건한 + 임팩트 + 스펙 명확
- **종결어미**: 정중 또는 트렌디 (~입니다 / ~예요)
- **권장**: 수치·벤치마크·인증·호환성

## Beauty (뷰티)
- **프레임워크**: FBM (Pleasure + Hope) + AIDA
- **톤**: 부드러운 + 감각적 + 디테일
- **권장**: 텍스처 묘사·향·비주얼

## Fashion (패션)
- **프레임워크**: AIDA + 4U
- **톤**: 트렌디 + 라이프스타일
- **권장**: 코디·시즌·취향

# 섹션별 카피 패턴 (스타일_1 표준)

## Hero
```
[Eyebrow]      한정·시즌·카테고리 라벨 (예: "5-6월 제철 한정")
[H1 (3 lines)] Pleasure spark — 즉시 감각 자극 (예: "칼 대는 순간, 집 안 가득 퍼지는 꿀향.")
[Subtitle]     상품 정보 1줄 (예: "국내산 고령 성산 메론, 4종 중 2~3종을 한 박스에")
```

## Hook — Fear/Hope 듀얼
```
[Eyebrow]      "WHY NOW"
[H1]           "놓치면 [내년까지] 기다립니다" (accent로 강조)
[Subtitle]     맥락 1줄

[Fear card]
  · label icon: ⏰
  · label: "FEAR · 손실 회피"
  · meta: "1년에 단 1번 기회"
  · h2: "성산 메론은 / 5~6월이면 끝."
  · highlight band: "지금 이 당도, 1년에 단 1번뿐인 기회."
  · footer arrow: "▶ 수확 직후 출고 — 이번 주에 딴 걸 이번 주에"

[Hope card]
  · label icon: 💛
  · label: "HOPE · 미래 장면"
  · meta: "택배 열자마자..."
  · h2: 장면 묘사 2줄
  · quote intro: "한 조각 집어든 가족이 —"
  · quote (italic serif): "\"이거 뭐야, 진짜 맛있네!\""
  · quote footer: "— 첫 한 입 그 순간"
```

## Taste
```
[Eyebrow]      "TASTE"
[Intro serif]  "반으로 가른 첫 순간,"
[H1 (italic)]  "\"이 맛은, 설명이 안 되네…\""
[사진]         단면 사진 (520×320 rounded)
[3 step cards] ① 첫 입 / ② 중간 / ③ 마무리
[Closing]      대비 강조 (수입산 vs 우리)
```

## Macro / Size
```
[풀와이드 사진]
[Highlight band] 정량 정보 1줄 (예: "📏 3kg 박스 — 평균 2~3알")
```

## POINT 01~04
```
[Eyebrow]      "POINT 0N"
[H1 with accent]
[Subtitle]
[메인 사진]
[혜택 카드 또는 chip 그리드]
[Footnote]     ※ 주의/제한 사항
```

## Cuts
```
[Eyebrow]      "CUTS"
[H1]
[Subtitle]
[좌우 큰 사진 2장]
[좌우 캡션]    "▶ 통+단면 모음" / "▶ 4종 컬러 비교"
[Footer quote] "이 색·이 향, 한 박스에 다 들어있어요."
```

## Kairos
```
[Eyebrow]      "RIGHT NOW"
[H1]           "수확 [이번 주] · 출고 내일"
[Body]         후숙·제철 설명 2줄
[Brand orange card] CTA punch (이번 주에 딴 걸, 이번 주에)
[Social quote] "식탁에서 들리는 한 마디 —"
[Social punch] "\"이거 어디서 샀어?\""
```

## CTA
```
[Eyebrow pill (white on dark)] "SUMMER LIMITED"
[Title 2 lines]
[옵션 ①·② 흰 카드]  가격 + 정가 + 할인율
[강조 띠]      "⏰ 수확 이번 주 · 출고 내일 · 무료배송"
[배송 안내 3줄]
[Closing italic]
[Brand]        달콤살랑 · DALKOM SALANG
```

`~/.claude/projects/c--antigravity-claude-code/memory/MEMORY.md` 의 다음 규칙은 모든 카피에서 강제:

### feedback_copy_factcheck.md
> 산지·농가·재배 조건 단정 금지, 팩트 없으면 감성 일반화 표현으로

- ❌ "100% 유기농", "특허받은", "유일한", "최고의 효능"
- ❌ "지난 30년간 1등", 검증 안 된 비교
- ✅ "촉촉한 식감", "은은한 단맛" (감각 일반화)
- ✅ "노란 껍질·흰 과육" (관찰 가능)

### feedback_hero_photo.md
> 히어로 사진은 항상 과육 단면/과즙 보이는 식욕 자극 사진

식품 카테고리에서 히어로는 무조건 잘려진 단면 또는 식욕 자극 컷.

### feedback_pricing_disclosure.md
> 가격 제안 시 4개 지표 동시 공개 (마진/마진율/수수료포함마진/수수료포함마진율)

가격 분석 시 카피라이터가 직접 다루지 않지만, 디렉터가 사용자와 가격 논의 시 적용.

### feedback_scope_discipline.md
> 사용자 승인 범위만 수정, 요청 외 항목 임의 변경 금지

카피라이터는 브리프와 사용자 컨펌 범위 안에서만 작성.
