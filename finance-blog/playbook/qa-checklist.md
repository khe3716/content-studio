# QA 체크리스트 (월급쟁이 재테크)

## CRITICAL — 1개라도 실패 시 발행 차단

- [ ] 모든 금리·한도·조건에 출처 표기 ("출처: 은행연합회 / 2026-04-25 기준")
- [ ] 정치인·정당·이념 발언 0
- [ ] 부동산 가격 전망 0
- [ ] 주식 종목 추천 0
- [ ] 단정 표현 0 ("100% 승인", "원금 보장", "확실한 수익")
- [ ] 사장님 개인 재무·세무 상담 톤 0 (자격 없음)
- [ ] 어필리에이트 사용 시 광고 명시 ("이 글은 제휴 마케팅을 포함합니다")

## HIGH — 3개 이상 실패 시 재작성

- [ ] 박재은 페르소나 톤 일관 (친근·뉴스 도입)
- [ ] 메인 키워드 제목·H1·도입에 포함
- [ ] 메타 설명 150자 이내
- [ ] 모든 이미지에 alt 존재
- [ ] 영상 후킹 5초 강력
- [ ] 영상 자막 가독성 (외곽선·크기)
- [ ] 영상 종횡비 정확 (롱폼 16:9, 쇼츠 9:16)

## MEDIUM — 통과하지만 다음 글에 반영

- [ ] 이모지 1~2개 (과다 X)
- [ ] 문장 평균 25자 이하
- [ ] 컬러 팔레트 준수
- [ ] 표·차트 모바일 가독성

## 자동 검증 가능 항목

| 항목 | 검증 방법 |
|---|---|
| 정치 키워드 금지 | 본문 grep `대통령|정당|보수|진보|좌파|우파` |
| 단정 표현 금지 | 본문 grep `100%\|원금 보장\|확실` |
| 출처 누락 | 표·랭킹 섹션에 `출처:` 키워드 존재 여부 |
| 이미지 alt | HTML `<img>` 태그 alt 속성 검사 |
| 메타 설명 길이 | 150자 |
| 영상 종횡비 | ffprobe로 width/height 확인 |
| 영상 길이 | 롱폼 55~65s, 쇼츠 28~32s |
