# 보험 베스트 프랙티스

## 핵심 키워드
- 메인: `실손보험 비교`, `암보험 추천`, `운전자보험`, `어린이보험`
- 시즌: 12월 (연말정산 + 보험료 갱신)
- 롱테일: `4세대 실손`, `갱신형 vs 비갱신형`, `보험료 절약`

## 글 구조
1. 도입 (보험 트렌드 뉴스 — 실손 4세대 의무화 등)
2. 본인 필수 vs 선택 정리 (실손 + 암 + 운전자 = 3대 필수)
3. 비교 표 (회사 / 보장 / 보험료 / 갱신 주기)
4. 시나리오 (30대 직장인 / 40대 가장 / 50대 노후 대비)
5. 체크 포인트 (병력 / 갱신·비갱신 선택 / 보장 한도)

## 정책·심의 (CRITICAL)
- 보험 광고는 **금융소비자 보호법** 적용
- "최고", "1위", "유일한" 단정 표현 금지 (광고 심의 위반)
- 의료 행위·질병 진단 단정 금지
- "이 보험만 들면 안전" 같은 표현 0
- 어필리에이트 사용 시 "광고" 명시 의무

## 어필리에이트 매칭 (단가 2~10만원/건)
- 굿리치 / 보맵 / 인슈로보
- DBDBDEEP 보험 비교
