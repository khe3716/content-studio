# 적금·예금 베스트 프랙티스

## 핵심 키워드

- 메인: `고금리 적금`, `1금융권 적금`, `직장인 적금`
- 시즌 (월별 갱신): `2026년 N월 고금리 적금 TOP 10`
- 비교: `토스뱅크 적금`, `카카오뱅크 자유적금`, `청년도약계좌`
- 롱테일: `월 30만원 적금`, `1년 만기 적금`, `우대 조건 없는 적금`

## 시즌 캘린더

| 월 | 검색 강세 토픽 |
|---|---|
| 1월 | 새해 재테크 첫 적금, 청년도약계좌 |
| 2~3월 | 신학기 자녀 적금, 사회초년생 |
| 5월 | **연중 최고점** — 이사·휴가 자금 적금 |
| 9월 | 추석 이후 결산 |
| 12월 | 연말정산 + 내년 적금 계획 |

## 글 구조 권장

1. **도입** — 한국은행 기준금리 / 시중은행 평균 금리 인용
2. **랭킹 표** — 순위 / 은행 / 적금명 / 최고금리 / 기본금리 / 한도 / 우대조건
3. **1위 강조** — 왜 1위인지 + 가입 조건 자세히
4. **체크 포인트 3가지** — 우대조건 충족 가능성 / 한도 / 만기
5. **시나리오** — "월 30만원 1년 / 월 50만원 2년" 시뮬레이션
6. **정리** — 다음 달 갱신 안내

## 후킹 카피 예시

- "5월에 적금 가입하면 늦었을까? 정답은…"
- "이 적금 안 보면 손해입니다 — 5월 TOP 10"
- "월 30만원, 1년 후 통장 잔고 차이"

## 영상 패턴

### 롱폼 60초
- 0~5s: 후킹 ("5월 적금 TOP 10 한 번에")
- 5~10s: TOP 10 미리보기 (작은 글씨로 빠르게)
- 10~50s: 10위→1위 카운트다운, 각 4초 (은행+금리+우대조건)
- 50~57s: 1위 강조
- 57~60s: 블로그 링크 CTA

### 쇼츠 30초
- 0~3s: 후킹 ("적금 TOP 5만 빠르게")
- 3~27s: 5위→1위, 각 4.8초
- 27~30s: 블로그 URL

## 데이터 소스 (사실 검증용)

- **은행연합회 소비자포털** — 금리 비교
- **금융감독원 파인** — 정확한 상품정보
- **각 은행 공식 사이트** — 우대조건 최신
- **한국은행 ECOS** — 기준금리·통계
