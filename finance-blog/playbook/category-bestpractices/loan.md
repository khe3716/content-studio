# 대출 베스트 프랙티스

## 핵심 키워드
- 메인: `마이너스통장`, `신용대출 금리비교`, `전세자금대출`, `대환대출`
- 시즌: 6월 (이사·휴가 자금)
- 롱테일: `직장인 신용대출`, `마이너스통장 한도`, `정부지원 대출`, `청년 전세자금`

## 시즌 캘린더
- **3월·9월** = 이사 시즌, 전세대출 강세
- **6월** = 연중 최고점, 휴가·여름 가전 구매
- **12월** = 연말 자금·내년 자금 계획

## 글 구조
1. 도입 (기준금리·은행 평균 금리 뉴스)
2. 비교 표 (은행 / 한도 / 금리 / 자격)
3. 추천 시나리오 (직장인 / 자영업 / 신혼 / 청년)
4. 체크 포인트 (DSR / 신용점수 / 우대조건)
5. 정부지원 대출 비교 (디딤돌·버팀목·청년)
6. 정리

## 금기 (대출 카테고리는 광고 심의 빡셈)
- "100% 승인" / "무심사" / "확정 대출" 0
- 부동산 가격 전망 0
- 특정 소득·신용점수 단정 ("○○점이면 ○○ 가능") 0
- "대부업·사금융" 절대 안내 X

## 어필리에이트 매칭
- DBDBDEEP 대출 비교
- 토스 대출 추천
- 핀크 / 뱅크샐러드
