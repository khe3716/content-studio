# 카피 패턴 (월급쟁이 재테크)

상위 재테크 블로그·유튜브 콘텐츠 분석을 통해 추출한 패턴 (MVP 버전, 운영하면서 보강).

## 1. 후킹 패턴 (도입 5초)

| 패턴 | 예시 | 효과 |
|---|---|---|
| 뉴스 짚기 | "오늘 한국은행이 ○○ 했죠. 그래서…" | 시의성·신뢰 |
| 손해 강조 | "이 적금 안 보면 손해입니다" | 공포·클릭 |
| 숫자 충격 | "월 30만원으로 1년 후 ○○만원" | 호기심 |
| 동질감 | "월급쟁이라면 누구나…" | 공감 |
| 질문 | "5월에 적금 가입하면 늦었을까?" | 답 찾기 |

박재은 메인 = **뉴스 짚기 + 동질감** 혼합.

## 2. 본문 구조 (블로그)

### 비교·랭킹형 (TOP N)
1. 도입 (뉴스 1건 + "그래서 우리는?")
2. 한 줄 요약 박스 (1위~N위 미리보기)
3. 본문 — 각 항목당 H2 또는 표
4. 1위 강조 박스
5. "가입할 때 체크할 3가지" 코너
6. 정리 + CTA

### 가이드·체크리스트형
1. 도입 (실패 사례 또는 뉴스)
2. "체크할 N가지" 미리보기
3. 각 항목 (H2 + 본문 3~5줄)
4. 정리 + CTA

### VS·비교형
1. 도입
2. 한 줄 결론 (선결론)
3. A vs B 표
4. 각 시나리오별 추천
5. 정리

## 3. 본문 톤 룰

- 문장 평균 25자 이하 (모바일 가독성)
- 긴 단락 금지 → 2~3문장씩 분리
- 핵심 수치는 `<span class="highlight">` 강조
- 출처는 `<span class="source">` 로
- "여러분" 대신 "이웃님" 또는 호칭 생략

## 4. CTA

| 위치 | 카피 | 목적 |
|---|---|---|
| 글 끝 | "이웃 추가하면 매주 정리 글 받아보실 수 있어요" | 재방문 |
| 카테고리 글 | "관련 글 → ○○" | PV 증대 |
| 어필리에이트 (애드센스 통과 후) | "○○ 가입하러 가기" | 전환 |

## 5. 영상 후킹 (5초)

- **롱폼 인트로**: "○○ 안 보면 손해입니다 / 5월 적금 TOP 10"
- **쇼츠 인트로**: "5월 적금 TOP 5만 빠르게" + 1위 미리 살짝 노출
- 첫 5초 안에 "왜 봐야 하는지" 명확
- 정적 이미지 금지 (모션·줌·페이드)

## 6. 영상 자막 룰

- 한 줄 최대 13자 (쇼츠) / 20자 (롱폼)
- 핵심 단어 컬러 강조 (#f59e0b)
- 폰트: Bold + 외곽선 검정 (배경 무관 가독성)
- 위치: 하단 1/3 (롱폼) / 중앙 (쇼츠)
