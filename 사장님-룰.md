# 사장님 운영 룰 (필수 준수)

> 매 세션 시작 시 가장 먼저 이 파일을 읽고 시작할 것.
> 이 룰을 위반하면 사장님이 화나심. 절대 잊지 말 것.
> 출처: 메모리 + backup/STATE-*.md 전체 + 세션 대화 종합

---

## 🚨 절대 원칙

1. **사장님이 시킨 것만 건드린다.** 안 시킨 거 절대 X.
2. **자동 trigger 자제.** 사장님 명시 요청 시에만 실행.
3. **잘 모르면 묻고 진행.** 추측해서 만지지 말 것.
4. **사실 확인 안 된 정보 단정 금지.** (환각 사고 = 사장님 신뢰 파괴)

## 📝 콘텐츠 카피·디자인 규칙

### 카피 톤
- ✅ 감성·고객 시점·약속형
- ✅ 대비형 헤드라인 + 스토리텔링
- ✅ 긍정 표현 ("두 마리 토끼", "공평한 약속" 등)
- ❌ 직설·명령·기능 나열
- ❌ 단정적 투자 권유, 과장 부사

### 사실 단정 금지 (CRITICAL)
- ❌ 산지·농가·재배 조건 단정 (확인 안 됐으면 일반화)
- ❌ 도착일 단정 ("5/8 도착 X", 마감일·주문일만 OK)
- ❌ 외부 농가·브랜드명 사용 (사장님 본인 농가 X)
- ❌ 시간 추측 (스크린샷 시계 보고 "새벽이니~")
- ❌ 정부 정책 수치·기간 추측 (민생회복지원금 사고)

### 상세페이지 디폴트
- **매거진 스타일** (수박 V6 톤)
- 풀블리드 사진 + 박스/버튼 X + 우아한 타이포
- 카드뉴스 톤 X

### 사진 규칙
- 히어로: 과육 단면/과즙 (식욕 자극)
- 각 섹션 대표 이미지 1장
- 상세페이지 사진은 **사용자 제공만** (AI 생성·stock photo·다른 상품 X)
- 큐레이션은 Gemini Vision에 위임 (클로드는 글·레이아웃·코드)

### 파일명 규칙

## 🛒 네이버 스마트스토어 등록 규칙 (달콤살랑)

### 클로드가 만져야 할 것 (10가지만)
1. **카테고리 설정**
2. **SEO 노출명**
3. **대표판매가**
4. **옵션**
5. **상세페이지 등록**
6. **썸네일 + 추가 이미지**
7. **상품 주요 정보**: 브랜드(달콤살랑), 제조사(달콤살랑 협력사), 인증(없음 → `greenCertifiedProductExclusionYn: true`), 원산지(사장님이 알려주는 대로)
8. **상품정보제공공시** (사장님 "알아서" 권한)
9. **검색설정** (네이버 사전 등록된 키워드만)
10. **리뷰 혜택**: 텍스트 100원 / 사진·동영상 1,000원

### 클로드가 절대 안 만지는 것
- A/S 전화번호
- A/S 안내 (afterServiceInfo 전체)
- 발송정보
- 반품정보
- 배송비
- 출고지·반품지
- 사업자 정보
- 최소구매수량 (`minPurchaseQuantity`)
- 판매자 코드 (`sellerCodeInfo`)
- Page Title / Meta description (노출명만 사용)
- 소비자 상담번호 (food.customerServicePhoneNumber)
- 기타 사장님이 이미 설정한 모든 기본값

### 필수 필드인데 사장님이 안 채워주면
- 사장님 다른 상품 (살구 `originProductNo: 13459378619`) 값을 **그대로 복사**
- 사장님 본인 값이므로 "건드린 거" 아님
- A/S 전화번호 = `01077330874` 등

### 임시저장
- NAVER 공식 임시저장 API **없음**
- 대안: `statusType: "SALE"` + `channelProductDisplayStatusType: "SUSPENSION"` (전시중지)
- 사장님 화면에서는 "판매중지" 탭에 보임 (진짜 "임시저장" 탭 X)

### 등록 후 잔재 청소
- origin product 삭제 시 channel product **자동 삭제 안 됨**
- `DELETE /v2/products/channel-products/{channelProductNo}`로 별도 청소
- "2개 상품"으로 보이면 잔재 채널 상품 확인 필수

### 카테고리 ID (자주 사용)
- 파인애플: `50002192`

### 원산지 코드
- 국산 (단순): `"00"`
- 태국: `"0200044"`
- 스페인: `"0201025"`
- (필리핀 코드 미확정 — 사장님이 알려주거나 관리자에서 수정)

### 단위가격 (살구 상품 구조)

  unitPriceYn: true,

### QA 검수 (박재은, publish 직전 자동 호출)
- `agents/finance/qa-reviewer.md` 페르소나로 Gemini 검수
- **정책 키워드** (지원금/보조금/민생/복지/장려금/쿠폰/환급/추경/정부지원/청년정책/소비쿠폰/재난지원/기초생활/차상위/주거급여/생계급여/의료급여/교육급여) 감지 시 엄격 모드
- critical 이슈 → DRAFT 유지 + 텔레그램 알림 + 사장님 검토 후 수동 발행

### 외부 자료 자동 수집

- **민감 정책 글**: 외부 자료 부족 시 환각 위험 → QA 차단

## 💬 사장님 페르소나·관계

- 사장님은 1인 스마트스토어 **달콤살랑** 운영 (과일 판매)
- **비개발자**, 한국어
- 핸드폰 텔레그램 명령 선호
- 컴퓨터 꺼져 있어도 자동화 작동 필요
- 애드센스 심사 중 — 과한 상품 링크 자제

### 사장님 자주 화내는 패턴 (= 절대 하지 말 것)
- 시키지 않은 거 만짐
- 같은 사고 반복
- 시간 끄는 답변
- 추측·환각으로 단정
- 핵심 안 짚고 빙 도는 설명
- 자동 trigger
- 디자인·정보 사장님 명시와 다르게

### 사장님 자주 만족하는 패턴
- 정확히 시킨 것만 처리
- 빠르고 짧은 답변
- 솔직한 한계 인정
- 실수 즉시 사과·수정
- 코드 변경 후 push 끝 보고
