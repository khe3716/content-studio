# 네이버 블로그 개설 가이드 (달콤살랑 유입용)

사장님이 직접 해야 하는 작업. 40분이면 끝.

## ⚠️ 시작 전 꼭 읽기

네이버 블로그는 **검색 지수(C-Rank, D.I.A)** 가 있어요. 지수가 높을수록 네이버 검색 상위에 올라가요.

### 절대 하지 말 것 (초기 3개월)

- ❌ 달콤살랑 스토어 링크 넣기
- ❌ "저희 매장에서 팝니다" 언급
- ❌ 외부 상업 블로그 링크
- ❌ 복붙 글 (네이버는 중복 콘텐츠 즉시 감지)
- ❌ 하루 2편 이상 발행 (저품질 판정)
- ❌ 키워드 억지로 많이 반복

### 대신 할 것

- ✅ 주 3~4편, 순수 정보성
- ✅ "저는 이렇게 해요" 체험 위주
- ✅ 사진 3장 이상 (가능하면 실사)
- ✅ 1,500자 이상

3개월 지나 지수가 쌓이면, 그때부터 후기 카테고리를 만들어서 5편 중 1편꼴로 상품 연계 글을 써요.

---

## 1단계: 계정 만들기 (5분)

1. [네이버 회원가입](https://nid.naver.com/nidregister.form) — 스마트스토어 계정과 **같은 아이디**로 하면 좋음 (나중에 연동 편함)
2. [블로그 개설](https://section.blog.naver.com) → 우측 상단 "내 블로그" → "블로그 만들기"
3. **블로그 주소(URL)** 는 한 번 정하면 못 바꿔요. 추천:
   - `dalkomsalang-fruit` (브랜드명 + fruit, 영문)
   - `fruitguide-korea` (일반 정보 느낌)
   - 달콤살랑 브랜드 묶이는 게 부담이면 중립 URL 추천

---

## 2단계: 프로필 + 스킨 (10분)

### 블로그 이름 / 별명
- 블로그 이름: **"과일정보연구소"** (과일 블로그 브랜드와 통일하는 게 좋음)
- 별명: **박과일** (또는 본명 X, 브랜드 유지)

### 프로필 소개
> 15년간 시장·산지·마트를 돌며 쌓은 과일 고르는 법과 보관법을 기록합니다. 제철 과일, 제대로 고르고 제대로 먹기.

### 스킨
- 기본 심플 스킨 선택 (꾸미기보다 **깔끔**이 지수에 유리)
- 폰트는 본고딕/나눔고딕 (기본)

### 프로필 사진
- 과일 사진 1장 (직접 찍은 거 or AI 생성)
- 로고 쓰지 말 것 (달콤살랑 브랜드 티 나면 상업 판정 위험)

---

## 3단계: 카테고리 구성 (5분)

관리 → 카테고리 관리 → **카테고리는 3개만** 만드세요.

| 카테고리 | 설명 | 발행 빈도 |
|---|---|---|
| 제철 과일 가이드 | 시즌별 고르는 법, 구별법 | 주 2편 |
| 과일 보관·손질 | 보관법, 손질법, 실패 안 하는 법 | 주 1편 |
| 과일 꿀팁 | 영양, 활용 레시피, 잘 아는 지식 | 주 1편 |

**후기 카테고리는 지금 만들지 말 것.** 3개월 뒤에 추가하면 돼요.

---

## 4단계: 첫 글 발행 (15분)

아래 순서로 하세요.

### (1) 텔레그램에서 명령어

```
/fruit
```

과일블로그 자동 발행이 돌아가요 (3~4분 기다림).

### (2) 변환 스크립트 실행 (사장님 컴퓨터에서 또는 다음 텔레그램 명령)

```
/naver
```

네이버용 HTML이 생성되고, 텔레그램에 **복사 링크**가 와요.

### (3) 복사 → 네이버 블로그 붙여넣기

1. 텔레그램의 HTML 링크 클릭 → 웹페이지 열림
2. **`<!-- ... -->` 주석 블록 아래**부터 **전체 선택 (Ctrl+A) → 복사 (Ctrl+C)**
3. 네이버 블로그 → 글쓰기
4. 제목 입력 (과일블로그 텔레그램 알림에 있는 제목 그대로)
5. 본문 입력창 **우하단 `</>` 버튼** 누르기 (HTML 편집 모드)
6. 붙여넣기 (Ctrl+V)
7. `</>` 버튼 다시 눌러서 일반 모드 복귀 — 이미지가 자동으로 뜸 (로딩 대기)
8. **대표 이미지 설정** — 본문 이미지 중 하나 우클릭 → "대표 이미지로"
9. **태그** — 카테고리 따라 3~5개 (예: `산딸기`, `과일보관법`, `제철과일`)
10. **발행** (공개)

### (4) 검색 노출 확인

- 발행 후 2~3일 지나 네이버에서 제목 검색 → 본인 글 노출 확인
- 처음엔 바로 안 나와요 (지수 0). **3주 정도 꾸준히 발행해야 조금씩 뜨기 시작**

## 5단계: 운영 규칙 (매일)

### 주 3~4편 페이스 유지

- 월/수/금 아침 `/fruit` → `/naver` → 복붙 발행 (글당 5분)
- 한꺼번에 몰아서 여러 편 발행하지 말 것 (저품질 판정)

### 댓글·이웃

- 첫 달: 다른 과일/식품 블로거 30명 이웃 맺기 (수동)
- 댓글 달린 글엔 24시간 내 답글

### 이미지 관리

- 자동 생성된 AI 이미지는 그대로 써도 OK
- 여유 있으면 실사 1~2장 추가 (배송 온 상품 사진 등)

### 지수 모니터링

네이버 공식 지수는 공개 안 되지만, 간접 지표:
- 검색 노출 시작 (3~4주차)
- 방문자 수 일 100명 (3개월차)
- 이웃 300명 (6개월차)

---

## 6단계: 3개월 후 (수익 단계)

지수가 쌓이면:

- "달콤살랑 후기" 카테고리 추가
- **5편 중 1편만** 상품 연계 글 ("산딸기 후기 — 제가 산지에서 직접 고른")
- 달콤살랑 스토어 링크는 글 하단에 1개만
- 나머지 4편은 계속 순수 정보성 유지 (지수 보호)

## 자주 묻는 질문

**Q. 블로거(구글)와 네이버 중복으로 써도 돼요?**
A. 내용 똑같이 쓰면 안 됨. `/naver` 변환 스크립트가 **표현·문장을 조정**해줘서 원글과 달라지긴 하지만, 혹시 모르니 제목은 약간 다르게 쓰세요 (예: Blogger "산딸기 보관법" / 네이버 "산딸기 오래 보관하는 3단계").

**Q. 네이버에 바로 올리면 구글이 카피라고 판단해요?**
A. 구글은 순서로 판단해요. Blogger(구글)가 먼저 발행되니 Blogger가 원본으로 인정됨. 네이버 블로그는 네이버 검색 엔진이 관리하니 따로 돌아감.

**Q. 블로그 이름 바꿀 수 있어요?**
A. 블로그 이름/별명은 바뀜. **URL만** 못 바뀜.

**Q. 태그 10개 넘어가도 돼요?**
A. 최대 30개지만, 과하면 스팸 판정. 5~10개 권장.
