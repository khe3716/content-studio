# 네이버 블로그 (과일정보연구소)

달콤살랑 스마트스토어 유입용 네이버 블로그 운영 공간.

## 구조

```
naver-blog/
├── drafts/        # auto-publish-naver.js 가 생성한 네이버 스마트에디터용 HTML
└── README.md      # 이 파일
```

## 어떻게 만들어지나

1. `auto-publish-fruit.js` (또는 텔레그램 `/fruit`) → `fruit-blog/drafts/*.html` Blogger 글 생성
2. `auto-publish-naver.js` (또는 텔레그램 `/naver`) → 위 HTML을 네이버 스마트에디터 호환 포맷으로 정리
3. 텔레그램 알림 (raw URL + 복사 안내)
4. 사장님이 링크 열고 HTML 복사 → 네이버 블로그 스마트에디터의 `</>` 모드에 붙여넣기

네이버는 공식 API가 없어서 마지막 단계만 사람이 함 (글당 1분).

## 운영 규칙 (초기 3개월)

**절대 금지** (지수 보호):
- 달콤살랑 스토어 링크
- 상품 판매 언급
- 외부 상업 블로그 링크

**발행 페이스**:
- 주 3~4편 (월/수/금 권장)
- 하루 2편 이상 금지 (저품질 판정)

**글 구성**:
- 1,500자 이상, 사진 3장 이상
- 제철 정보, 보관법, 선별법 위주
- "저는 이렇게 해요" 1인칭 체험 톤

3개월 뒤 지수가 쌓이면 "후기" 카테고리를 추가하고 5편 중 1편꼴로 상품 연계 글을 씀.
