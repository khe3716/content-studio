## 🎨 V6 매거진 스타일 — 핵심 원칙

### 절대 금지
- ❌ 둥근 파스텔 박스
- ❌ 쇼핑몰 옵션 버튼 (둥근 알약·강한 컬러)
- ❌ 컬러 블록 섹션 (라임/크림/파스텔 배경 전체)
- ❌ 카드 그리드 2x2 4개 박스
- ❌ "SECTION 01" 강조 라벨 (큰 폰트)

### 필수 적용
- ✅ **풀블리드 사진** (W=600 가로 100%) 섹션마다 적극 배치
- ✅ **얇은 매거진 라벨** (`S T O R Y` letter-spacing 4, 11px, 회색)
- ✅ **얇은 가는 라인 dividers** (0.6~0.8px)
- ✅ **사진 위 텍스트**: linear gradient darken (위 0.25 → 아래 0.55~0.7)
- ✅ **거대 타이포**: BRIX 200px, 헤딩 32~36px
- ✅ **옵션 리스트**: 텍스트 + 라인 (BEST는 굵게 + 빨강)
- ✅ **딥그린 푸터** 마무리 (#1A3A1A + 흰 텍스트)

### 컬러 팔레트
```
화이트         #FFFFFF
페이퍼톤       #FBFAF7
와인레드       #6B1F1F  (메인 강조)
브랜드레드     #C53030  (BEST 옵션)
브랜드그린     #1B4D1C  (스토리지 강조)
딥그린         #1A3A1A  (푸터 배경)
다크텍스트     #1A1A1A
서브텍스트     #666666
뮤트          #999999
라인          #E5E5E5
```

### 7섹션 표준 구조
1. **HOOK** — 풀블리드 macro + gradient overlay + 흰 텍스트
2. **STORY** — 풀블리드 야외/현장 + 페이퍼 텍스트 + 키워드 라인 라벨
3. **QUALITY/STANDARD** — 3 원형 사진 (140px) + 18px 타이틀 + 13px 설명
4. **BRIX/USP** — 풀블리드 macro + 거대 수치 중앙 (200px)
5. **SIZE & OPTION** — 비교 사진 + 포장 사진 연달아 + 옵션 텍스트 리스트 1번만
6. **STORAGE & RECIPE** — 2x2 갤러리 (rounded square) + 캡션
7. **FAQ + FOOTER** — Q&A 매거진 텍스트 + 딥그린 푸터 블록
