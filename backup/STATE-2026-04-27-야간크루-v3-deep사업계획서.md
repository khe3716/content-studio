## 🐛 알려진 문제 / 미해결

### 1. 시간 들쑥날쑥 (가장 큰 문제)
- 현상: GitHub Actions 무료 cron 큐 지연 평균 +9시간 (±3시간)
- 결과: 약속 KST 08:00 도착 → 실제 평균 KST 14~17시 도착
- 사장님 표현: "계속 낮에 보내"

**해결 옵션** (사용자 결정 대기):
- **A. cron 시간만 조정** (1분 작업, 무료): `0 22 * * *` → `23 14 * * *` UTC (전날 KST 23:23 시작). 평균 도착 KST 09:00 ± 3h (06~12시 사이 들쑥날쑥은 유지)
- **B. 외부 cron (cron-job.org) + GitHub PAT**: 5분 작업, 무료, **KST 09:00 ± 10분 정확**
- **C. 그냥 지금처럼** (사용자가 임시 선택: "귀찮다") — 추가 작업 X, 도착 계속 오후

마지막 사용자 답변: **"시간조정은 가능하다고?"** — 옵션 A 고려 중.

### 2. 광고 게시 이해
사장님이 "사이트에 광고 어떻게 게시?"에 깊이 의문 가졌음. 핵심 정리:
- 구글 애드센스 + 카카오 애드핏 (자동 사진 광고, 심사 1~3개월)
- 쿠팡 파트너스 (제휴, 1주 자동 승인)
- 본인 상품 배너 (즉시 가능, 심사 X)
- 직접 영업 (트래픽 일 500명+ 후)

사장님 결론: **광고 가능성에 대해 확실히 이해됨**.

### 3. PDF 안 열림 (Antigravity IDE)
사장님이 reports/*.pdf 클릭으로 안 열린다고 함. Windows 기본 뷰어 실행 시도. **텔레그램 발송이 가장 확실** (핸드폰에서 보기). 미해결.
