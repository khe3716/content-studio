## 새 시스템 (이전 세션 대비)

### 1. AI 오디오 자동화 (3종)
- **Lyria 3 Clip BGM** (`scripts/finance-team/generate-bgm.js`)
  - Gemini SDK로 30초 instrumental 자동 생성
  - 모델: `lyria-3-clip-preview`
  - 출력: `finance-blog/remotion/public/audio/bgm/`
- **ElevenLabs SFX API** (`scripts/finance-team/generate-sfx.js`)
  - 텍스트 → 효과음 자동 생성 (4가지 + 20가지 옵션 풀 생성)
  - 무료 플랜 매월 10,000 크레딧
  - 환경변수: `ELEVENLABS_API_KEY`
  - 단, ElevenLabs가 가끔 묵음 가까운 SFX 생성 (rank_pop -46dB 사례) → 음량 검증 필수
- **Gemini Video API**
  - YouTube URL or 업로드 mp4 → SFX 시점·톤 자동 분석
  - `models/gemini-2.5-pro` + fileData
  - 사용 사례: 사용자 참고 영상 분석 → 효과음 8개 시간순 매칭

### 2. 사용자 효과음 적용
- 사용자가 8종 wav 파일을 `finance-blog/remotion/public/audio/sfx/user/` 폴더에 업로드
- 영상 시간순 1:1 매칭 (영상 SFX 순서 = 파일 번호 순서)
- 영문 짧은 이름 복사 (한글·공백·점 staticFile 호환 X)

### 3. 영상 명명 규칙 변경
