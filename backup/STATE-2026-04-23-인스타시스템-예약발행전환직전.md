| 🍎 과일 | 18:00 | 하루 1회 |

### 구현 요구사항
1. **DRAFT 업로드 → Playwright 자동 → SCHEDULED 발행 예약** 파이프라인
2. 각 실행 시간에 맞춰 **publishDate** 자동 설정
   - 경제 오전: publishDate = 오늘 07:30 KST (실행 시각이 07:30이면 즉시 발행)
   - 경제 오후: publishDate = 오늘 17:00 KST
   - 과일: publishDate = 오늘 18:00 KST
3. 여러 Day를 **미리 DRAFT로 생성**해두고 각각 다른 날짜·시간에 예약 배정
   - 오늘 / 내일 / 모레 / 3일뒤 / 4일뒤 ... 배치
4. 수동 실행 시엔 즉시 발행 or 다음 슬롯 예약

### Blogger API
`POST /blogs/{blogId}/posts/{postId}/publish?publishDate=2026-04-23T18:00:00+09:00`
- publishDate 미래면 SCHEDULED
- publishDate 과거면 즉시 LIVE
