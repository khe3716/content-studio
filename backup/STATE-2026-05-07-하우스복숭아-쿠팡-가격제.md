- 쿠팡 수수료 10.6% (식품 일반 기준) 반영
- 깔끔한 천원 단위 가격

## 3. 쿠팡 입력 시 주의사항

### 3.1 가격 입력
- **할인전 가격 (정상가)**: 위 정상가 입력
- **판매가 (할인가)**: 위 판매가 입력
- 옵션별 절대가격 입력 (스마트스토어처럼 옵션가 ±X원 방식 아님)

### 3.2 배송
- **로켓배송 vs 일반배송** 정책에 따라 추가 수수료 다름
- 위 계산은 **일반배송 기준** (10.6%)
- 로켓배송 선택 시 마진율 다시 계산 필요

### 3.3 광고비
- 순마진율 25%는 **광고비 제외** 기준
- 쿠팡 광고 돌리면 추가 차감
- CPC 기준 ROAS 모니터링 필수
