## Key Technical Details

### Blogger API
- Cannot set permalink or search description via API
- Workaround: Include both in Telegram notification as copy-paste strings
- User manually pastes in Blogger UI (10 sec per post)

### AI Image Generation (Fruit Blog Only)
- **Imagen 4 Fast** via Gemini API
- Prompts generated by `gemini-2.5-flash` with thinking disabled
- Output: 1024x1024 → resized to 600x600 JPG (smaller file size)
- Why smaller: Blogger picks largest image as featured thumbnail.
  Body images must be smaller than 1500x1500 designed thumbnail.

### Cache Busting
- **All thumbnails + body images use timestamp in filename**:
  - Thumbnail: `day-01-{Date.now()}.png`
  - Body: `day-01-{Date.now()}-body-1.jpg`
- Prevents Blogger CDN from caching old versions across re-uploads.

### Thumbnail Design
- **Economy**: 800x800 PNG, orange/cream gradient, Fluent 3D emoji composited
- **Fruit**: 1500x1500 PNG, red/pink text design, no emoji composite
  - Title font scales by length: 170/130/100/80
  - "FRUIT GUIDE" label top-left, "과일정보연구소" bottom-right

### Naver Commerce API Integration
- Client ID: stored in GitHub Secret `NAVER_COMMERCE_CLIENT_ID`
- Client Secret: `NAVER_COMMERCE_CLIENT_SECRET` (bcrypt salt format)
- Signature: HMAC-like with bcrypt
- Endpoint: `api.commerce.naver.com/external/v1/products/search`
- Fetches 21 products, saves to `fruit-blog/products.json`
- Used for context only (not for body images)

## Iteration History (Day 1 Fruit Blog)

The first fruit blog post (Day 1 산딸기) went through 7 iterations:
1. Green gradient thumbnail design (first attempt, wrong style)
2. .env missing error in CI → fixed fallback
3. Korean fonts missing → installed fonts-noto-cjk
4. Thumbnail cached via raw URL → private repo 404 → made public
5. Designed thumbnail too small → bumped to 1500x1500
6. Body images were Gemini-hallucinated Unsplash URLs → switched to Imagen 4 AI generation
7. Admin list showed wrong thumbnail (Blogger picked larger body image) → shrunk body images to 600x600 JPG
8. Browser cache showed old thumbnails → added timestamp to filenames
9. Thumbnail text too small → increased title font 30%

Final working state: All above issues fixed, Day 1 displays correctly with designed thumbnail as featured + 2 AI fruit photos in body.
