## 모션 규칙 (메모리 박힘)

### 1. 좌우 안전 마진 — 최소 20px
모바일 cropping 방지. ShortForm padding 90px (= 70 + 20 마진).

### 2. 메인 텍스트 ease-in / 루프 모션 sin/cos 그대로
큰 텍스트는 ease-in 임팩트, 떠다니는 도형은 sin/cos 루프 OK.

### 3. 글자 등장 풍부하게 (단순 fade·slide X)
StaggerTextRich (blur+scale+rotate+slide) 또는 BouncyDampedText (AE damped oscillation).

### 4. 모션 위계 (한 페이지 같은 ease-in 촌스러움)
- Hero (큰 텍스트): AE Bouncy 또는 StaggerTextRich
- Big Data (숫자): spring (살짝 bounce)
- 중간 텍스트: spring·ease-out (자연스럽게 도착)
- Caption/Meta (작은 라벨): 정적 또는 짧은 fade

→ 메모리: `feedback_video_motion_rules.md`

## AE 익스프레션 통합

사용자가 알려준 AE expression:
```
delay = .04;
myDelay = delay * textIndex;
t = (time - inPoint) - myDelay;
if (t >= 0) {
  freq = 3; amplitude = 30; decay = 8.0;
  s = amplitude * Math.cos(freq*t*2*Math.PI) / Math.exp(decay*t);
  [s, s];
} else { value }
```

→ Remotion `BouncyDampedText` 컴포넌트로 1:1 변환
→ 사용처: Hook hero 타이포 ("5월 적금 / TOP 5"), 은행명 (토스뱅크 등)

## 직전 작업 (Style A v4 — AE Bouncy 적용)

`videos/demos/style-A-full-30s.mp4` 최신 상태:
